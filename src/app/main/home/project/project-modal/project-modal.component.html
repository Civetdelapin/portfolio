Infinite Torpor

[Work In Progress]

Explore procedurally generated dungeons, fight back and try to find a way out!

This was a personal project with my little brother (game artist).

I built the game engine from scratch (game loop, collisions detection, buttons...), and developed game mechanics (combat system, enemies behavior...). The game is procedurally generated based on hand-crafted rooms.

Made in C++ with only SDL2.
